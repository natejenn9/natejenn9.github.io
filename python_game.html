Python Infinite Running Game

The Python Infinite Running Game is a dynamic project combining technical and creative skills. The game features custom-designed pixel art and a physics engine, showcasing expertise in Python programming and the Pygame library. The goal was to create an engaging and visually appealing gaming experience.

A key aspect of the project was designing the physics system, which required applying mathematical concepts to ensure realistic in-game mechanics. The custom pixel art added a personal touch, making the game stand out as a unique and polished product. This project reflects the ability to combine technical proficiency with artistic creativity.

Tools Used:

  • Python
  • Pygame
  • Custom Pixel Art
  • Physics Engine Development
