GUI-Based Analytic Engine

The GUI-Based Analytic Engine was developed to streamline data analysis for businesses. The project required designing a versatile tool capable of adapting to different locales, languages, and visual themes based on contract requirements. It featured dynamic theming, localization support, and robust unit testing to ensure reliability under various conditions.

This project was a collaborative effort, with tasks managed through pseudo-Jira ticketing. Over multiple two-week sprints, the team worked on implementing key functionalities and delivering a polished product. Comprehensive unit tests were created using JUnit, ensuring the system handled edge cases and maintained performance.

Tools Used:

  • Java
  • JUnit
  • Git
  • Visual Studio Code
